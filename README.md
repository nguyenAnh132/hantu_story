#Hantu Story is a lightweight social networking platform built with Spring Boot using a microservices architecture.
It allows users to connect and interact through posts, likes, comments, and follows.

##Features
User registration & login
Create and manage posts
Like and comment on posts
Follow other users
Scalable microservices architecture

##Tech Stack
Java, Spring Boot
Microservices Architecture
REST API
Docker
